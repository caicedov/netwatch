# MVP Scope

## Features IN (Must-Have for MVP)

- Real-time multiplayer interactions
- Basic hacking mechanics
- Player progression system

## Features OUT (Intentionally Excluded)

- Advanced hacking tools
- Extensive character customization

## Prioritization Order for IN Features

1. Real-time interactions
2. Basic mechanics
3. Progression system

## Reasoning for Exclusions

Focusing on core gameplay ensures a solid foundation for future expansions.
